Built a two-part 3D vision project that reconstructs geometry from calibrated stereo views and synthesizes novel views with a NeRF-style neural rendering pipeline. The project combines classical geometry, depth estimation, point-cloud processing, positional encoding, ray sampling, and differentiable volumetric rendering.
