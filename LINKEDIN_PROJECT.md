# LinkedIn Project Entry

## Title

3D Reconstruction and Neural Rendering

## Role

Computer Vision / Deep Learning Developer

## Description

Built a machine perception project that combines classical 3D reconstruction with neural rendering. The first module reconstructs a scene from calibrated two-view stereo using rectification, patch-based disparity matching, depth recovery, and point-cloud filtering. The second module implements a NeRF-style rendering pipeline in PyTorch, including positional encoding, camera ray generation, stratified sampling, radiance-field prediction, and differentiable volumetric rendering.

## Highlights

- Implemented stereo rectification, SSD/SAD/ZNCC patch matching, left-right consistency checks, and depth-to-point-cloud projection.
- Built PyTorch modules for coordinate-based image fitting and NeRF-style novel-view synthesis.
- Worked with camera intrinsics/extrinsics, multi-view datasets, point-cloud processing, and neural volume rendering.
- Organized the project into a clean GitHub-ready repository with reusable source files, documentation, data, and setup instructions.

## Skills

Python, PyTorch, NumPy, OpenCV, Open3D, Computer Vision, 3D Reconstruction, Stereo Vision, NeRF, Neural Rendering, Volumetric Rendering, Camera Geometry
